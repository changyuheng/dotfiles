# Dotfiles by changyuheng

## Installation

### OS X

**Prerequisites**

- Xcode and its Command Line Tools
- CMake

**Do install**

```
./os-x-install.sh
```

### Ubuntu

**Prerequisites**

```sh
sudo apt-get install build-essential cmake python-dev git
```

**Do install**

```sh
./ubuntu-install.sh
```

## Uninstallation

### OS X

```
./os-x-uninstall.sh
```

### Ubuntu

```sh
./ubuntu-uninstall.sh
```
